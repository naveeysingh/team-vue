# Vue Notes App

A simple notes application built with Vue.js. This app allows you to create, read, update, and delete notes. It features a modern design with a welcome header, smooth transitions, and a fixed footer.

## Features

- **Create Notes:** Add new notes with a title and content.
- **Read Notes:** View a list of all created notes.
- **Update Notes:** Edit the content of existing notes.
- **Delete Notes:** Remove notes from the list.
- **Responsive Design:** A clean and modern UI that works well on various screen sizes.


## Files

The project consists of three main files:

- **index.html:** The main HTML file that sets up the Vue.js app and includes the header, main content, and footer.
- **styles.css:** The CSS file that provides the styling for the app.
- **script.js:** The JavaScript file containing the Vue.js code for the app's functionality.

## How to Run

1. **Download or Clone the Repository:**
   ```bash
   git clone https://github.com/naveeysingh/team-vue.git
   ```

2. **Navigate to the Project Directory:**
   ```bash
   cd vue-notes-app
   ```

3. **Open the `index.html` File:**
   You can open the `index.html` file directly in your browser by double-clicking it or by serving it with a local web server.

 
   Then, navigate to `http://localhost:8000` in your web browser.

## Dependencies

- [Vue.js 3](https://vuejs.org/) - The app uses Vue.js 3 via CDN.
